Altius Chat Widget Demo

Basic Usage

Add this to your HTML:

<script src="index.js" data-auto-init></script>

Advanced Usage

Initialize with custom configuration:

<script>
window.AltiusChatWidget.init({
    primaryColor: '#2563eb',
    textColor: '#ffffff',
    backgroundColor: '#f9fafb',
    borderRadius: '12px',
    welcomeMessage: 'Welcome to our support!'
});
</script>

Demo Controls

API Reference

// Get widget instance
const widget = window.AltiusChatWidget.init(config);

// Open chat
widget.open();

// Close chat
widget.close();

// Clear chat history
widget.clearHistory();
