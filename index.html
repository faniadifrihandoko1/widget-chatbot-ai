Altius Chat Widget Demo

Cara Penggunaan Widget

Untuk menggunakan Altius Chat Widget, tambahkan kode berikut ke halaman HTML Anda: 

<script src="https://altius-chat-bot.netlify.app/altius-chat-widget.min.js"></script>
<script>
  // Konfigurasi API
  window.chat_api_id = "YOUR_API_ID";    // ID API Anda
  window.chat_api_key = "YOUR_API_KEY";  // API Key Anda
  window.personal_data = "DATA_PERSONAL"; // Data personal pengguna

  
</script>
