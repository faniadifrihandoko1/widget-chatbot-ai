Altius Chat Widget Demo

Cara Penggunaan Widget

Untuk menggunakan Altius Chat Widget, tambahkan kode berikut ke halaman HTML Anda: 

<script src="https://cdn.altius.com/chat-widget/altius-chat-widget.min.js"></script>
<script>
  window.AltiusChatWidget.init({
    // Konfigurasi opsional
    primaryColor: '#2563eb',    // Warna utama widget
    textColor: '#ffffff',       // Warna teks
    backgroundColor: '#f9fafb', // Warna latar belakang
    borderRadius: '12px',       // Sudut melengkung
    welcomeMessage: 'Selamat datang! Ada yang bisa kami bantu?' // Pesan selamat datang
  });
</script>

Metode yang Tersedia

// Mendapatkan instance widget
const widget = window.AltiusChatWidget.init(config);

// Membuka chat
widget.open();

// Menutup chat
widget.close();

// Membersihkan riwayat chat
widget.clearHistory();
