Altius Chat Widget Demo

Cara Penggunaan Widget

Integrasi mudah & cepat ke website Anda
  1. Salin dan tempelkan kode berikut ke dalam <head> atau sebelum </body> di HTML Anda.
  2. Ganti YOUR_API_KEY dan DATA_PERSONAL sesuai kebutuhan.
  3. Widget siap digunakan! (Opsi: sesuaikan tampilan dengan konfigurasi tambahan)
  4. Klik ikon chat di pojok kanan bawah untuk mulai bertanya.
<script src=\"https://altius-chat-bot.netlify.app/altius-chat-widget.min.js\"></script>
<script>
  // Konfigurasi API
  window.session_id = \"YOUR_SESSION_ID\";    // opsional
  window.chat_api_key = \"YOUR_API_KEY\";  // wajib
  window.personal_data = \"DATA_PERSONAL\"; // wajib

</script>
Keterangan:
wajib = Harus diisi  | DATA_PERSONAL = Data personal pengguna
